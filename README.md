# README

## 開発言語
* Ruby3.0.1
* Rails6.0.4
***
<br />

## 実行手順
```
$ git clone git@github.com:kashimakoji/origin_app.git
$ cd frugality
$ bundle install
$ rails db:create && rails db:migrate
$ rails s
```
